# **App Name**: BeachPal

## Core Features:

- Registration and Authentication: Secure registration and authentication system for beach tent owners and customers with email verification, CPF validation, full name, and address.
- Beach Tent Registration and Management: Beach tent owners can register their tents, upload photos to a gallery, and manage available umbrellas and chairs for rent.
- Menu Management: Beach tent owners can create and update their menu with prices in a news feed format.
- Real-time Location Tracking: Beach tent owners share real-time location so customers can easily find them on the map. In addition, Google Maps can initiate routes from the customer to the tent in real time.
- Online Ordering and Payment: Customers can view the menu, place orders online, and see the total bill. Tent owners are responsible for cancellation requests.
- Map View and Tent Discovery: Initial full-screen Google Maps display showing registered beach tents. Users can click to view details and make reservations.
- Table and Chair Reservation: Customers can reserve tables and chairs (or pre-defined kits) directly through the app, and complete rentals with payment.

## Style Guidelines:

- Primary color: Vibrant turquoise (#40E0D0), reminiscent of clear tropical waters.
- Background color: Very light cyan (#E0FFFF), provides a clean, airy feel.
- Accent color: Yellow-orange (#FFB347) provides contrast and highlights key actions.
- Body and headline font: 'PT Sans', a humanist sans-serif providing both modern appeal and readability.
- Simple, outlined icons representing beach-related activities (sun, umbrella, drink, etc.).
- Clean and intuitive layout with a focus on map integration and easy navigation.
- Smooth transitions and subtle animations for a seamless user experience.
